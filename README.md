### âš™ï¸ Configuration
It's Fully Configurable! Change everything to suit your playstyle. Thanks to seamless integration with **YetAnotherConfigLib** and **Mod Menu**, you have total control over the mod's behavior:
- Define custom **Burn Sources** (decide which items act as a "trash can").
- Control **Resource Usage**: Decide if fire sources like Flints and Steels or Fire Charges should lose durability/be consumed, or if they can be used as infinite trash cans.
- Manage the **Protected List** for custom item safety.
- Fine-tune **Audio Settings** like volumes and pitches for that perfect crunchy burn sound.
